# 🚀 НАЧНИТЕ ЗДЕСЬ!

## Ваша цель: Настроить GitHub Pages для VLESS конфигов

**Время настройки:** 5-7 минут

---

## 📋 3 простых шага

### Шаг 1️⃣: Создайте репозиторий на GitHub (2 мин)

1. Откройте: https://github.com/new
2. Название: `vless-configs`
3. **Public** (публичный) ✅
4. Create repository

👉 **Дальше:** Откройте [`QUICK_START_PAGES.md`](./QUICK_START_PAGES.md)

---

### Шаг 2️⃣: Заполните .env файл (2 мин)

1. Откройте файл `.env` в проекте
2. Вставьте свои токены
3. Сохраните

👉 **Помощь:** Откройте [`ENV_SETUP_GUIDE.md`](./ENV_SETUP_GUIDE.md)

---

### Шаг 3️⃣: Проверьте работу (1 мин)

1. Запустите бота: `python bot/main.py`
2. Нажмите "Get Proxy" в Telegram
3. Проверьте GitHub Pages

👉 **Проверка:** Откройте [`PAGES_CHECKLIST.md`](./PAGES_CHECKLIST.md)

---

## ❓ Что-то не получается?

1. Проверьте, что репозиторий **Public**
2. Убедитесь, что токены скопированы без ошибок
3. Подождите 2-3 минуты после первого коммита
4. Откройте **GITHUB_PAGES_SETUP.md** раздел "Troubleshooting"
