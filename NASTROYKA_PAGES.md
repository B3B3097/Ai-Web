# 🚀 НАСТРОЙКА GITHUB PAGES - ПОШАГОВАЯ ИНСТРУКЦИЯ

## ⏱️ Время настройки: 5-7 минут

---

## ШАГ 1: Создайте репозиторий на GitHub (2 мин)

1. Откройте https://github.com/new
2. Заполните поля:
   - **Repository name:** `vless-configs`
   - **Description:** `VLESS Configuration Storage`
   - ✅ **Public** (обязательно публичный для Pages!)
   - ❌ Не инициализируйте README
3. Нажмите **Create repository**

---

## ШАГ 2: Включите GitHub Pages (1 мин)

1. В созданном репозитории перейдите в **Settings** → **Pages**
2. В разделе **Build and deployment**:
   - **Source:** Deploy from a branch
   - **Branch:** main → / (root)
3. Нажмите **Save**
4. Подождите 1-2 минуты
5. Скопируйте ваш URL (примерно): 
   ```
   https://ВАШ_USERNAME.github.io/vless-configs/
   ```

---

## ШАГ 3: Создайте GitHub Token (2 мин)

1. Откройте https://github.com/settings/tokens
2. Нажмите **Generate new token** → **Generate new token (classic)**
3. Заполните:
   - **Note:** `vless-bot-pages`
   - **Expiration:** No expiration (или выберите срок)
   - ✅ **repo** (полный доступ к репозиторию)
4. Нажмите **Generate token**
5. **СКОПИРУЙТЕ ТОКЕН** (больше не увидите!)

---

## ШАГ 4: Настройте .env файл (1 мин)

Откройте файл `.env` и заполните:

```bash
# GitHub Token (из шага 3)
GITHUB_TOKEN=ghp_XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

# Имя репозитория (ОДИН и тот же для конфигов и Pages)
CONFIG_REPO=ВАШ_USERNAME/vless-configs
PAGES_REPO=ВАШ_USERNAME/vless-configs

# Другие настройки...
```

**Важно:** Замените `ВАШ_USERNAME` на ваш логин GitHub!

---

## ШАГ 5: Запустите бота

```bash
python bot/main.py
```

При первом запуске бот автоматически:
- ✅ Создаст папку `configs/` в репозитории
- ✅ Создаст файл `.gitkeep` для отслеживания директории
- ✅ Загрузит первый конфиг

---

## ШАГ 6: Проверьте результат

1. Откройте ваш Pages URL: `https://ВАШ_USERNAME.github.io/vless-configs/`
2. Вы должны увидеть таблицу с загруженными конфигами
3. Каждый конфиг доступен по ссылке вида:
   ```
   https://ВАШ_USERNAME.github.io/vless-configs/configs/username_20250101_120000.txt
   ```

## 🔧 ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### ❌ Страница не открывается (404)
**Решение:** Подождите 2-5 минут после включения Pages. GitHub обрабатывает запрос.

### ❌ Ошибка аутентификации
**Решение:** Проверьте токен в `.env`. Убедитесь, что у токена есть права `repo`.

### ❌ Конфиги не загружаются
**Решение:** Убедитесь, что репозиторий существует и у токена есть доступ.

### ❌ index.html не обновляется
**Решение:** Проверьте логи бота. Файл обновляется автоматически при загрузке конфига.

## 🎯 ЧТО ПРОИСХОДИТ АВТОМАТИЧЕСКИ

После настройки бот будет:

1. **При загрузке конфига:**
   - Создавать файл `configs/username_timestamp.txt`
   - Обновлять `index.html` со списком всех конфигов
   - Сохранять UUID при обновлении существующего конфига

2. **GitHub Pages:**
   - Автоматически публиковать изменения через 1-2 минуты
   - Предоставлять публичный доступ к конфигам

3. **Структура репозитория:**
   ```
   vless-configs/
   ├── configs/
   │   ├── .gitkeep
   │   ├── user1_20250101_120000.txt
   │   └── user2_20250101_130000.txt
   └── index.html (автогенерируемый)
   ```
